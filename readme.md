
# ☕ Coffee Shop Sales Dashboard – MS Excel Project

# 📌 Project Overview

This project focuses on analyzing coffee shop sales data using Microsoft Excel. An interactive dashboard was developed to monitor sales performance, product trends, customer purchasing behavior, and store-wise revenue. The dashboard transforms raw transaction data into meaningful business insights, enabling data-driven decision-making and operational improvements.

---

# 🎯 Project Objectives

* Analyze sales and transaction performance.
* Identify top-performing products and categories.
* Compare revenue across store locations.
* Track monthly sales trends and customer demand.
* Create an interactive dashboard for business reporting.

* Certain product categories consistently outperformed others.
* Revenue trends aligned closely with transaction volume.
